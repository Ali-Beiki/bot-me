## Installation
```sh
 npm i
```

## Edit config file
_Setting the environment variable_
- TOKEN="YOR_TOKEN"
- ADMIN_ID=ADMIN_ID

## Note
The express package was for receiving https requests so that the robot does not fall asleep on the glitch, and if you buy a host, you do not need express!
